## What is EmberScan?

EmberScan is a comprehensive firmware security analysis tool designed for:

- **Security Researchers** - Discover vulnerabilities in IoT and embedded devices
- **Penetration Testers** - Assess device security during engagements
- **Device Manufacturers** - Test firmware before release
- **Bug Bounty Hunters** - Find and report security issues

## Key Features

| Feature | Description |
|---------|-------------|
| Firmware Extraction | Extract filesystems from SPI dumps and manufacturer files |
| QEMU Emulation | Emulate firmware for dynamic analysis |
| Vulnerability Scanning | Comprehensive security checks |
| CVE Correlation | Match findings against known vulnerabilities |
| Professional Reports | HTML, JSON, and SARIF output formats |

## Supported Architectures

- ARM / ARM64 (AArch64)
- x86 / x86_64
- PowerPC
